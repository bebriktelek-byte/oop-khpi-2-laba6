Лабораторна робота № 6: Поліморфізм з абстрактними класами в JavaВиконавецьСтудент: Ткаченко ТимурГрупа: КН-1024ВВаріантНомер: 9 Тема: Страхові поліси (Insurance Policies) Опис ієрархії класівВідповідно до індивідуального завдання, реалізовано ієрархію для моделювання страхових послуг:Абстрактний базовий клас: InsurancePolicy — містить спільний стан (номер полісу, ПІБ власника, дата завершення) та оголошує спільний поліморфний контракт.Конкретні підкласи:HealthPolicy — медичне страхування.VehiclePolicy — страхування транспортних засобів.TravelPolicy — страхування для подорожей.Що продемонстровано в роботіАбстракція: Виділено узагальнене поняття страхового полісу, яке неможливо створити безпосередньо (використано abstract class).Поліморфний контракт: Оголошено абстрактний метод (наприклад, calculatePremium() або getCoverageDetails()), який перевизначено в кожному підкласі за допомогою @Override.Динамічна диспетчеризація: У класі Main продемонстровано роботу з масивом типу InsurancePolicy[], де для кожного об'єкта викликається специфічна логіка через посилання базового типу.Тестування: Додано JUnit-тести для перевірки коректності розрахунків та коректного виклику методів через базовий тип.
